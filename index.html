Chosen Feature: CSS Nesting

Introduction to CSS Nesting

What is CSS Nesting?:

CSS nesting is a new way to organize your CSS code, that helps reduce the redundancy of repeated selectors in CSS. Before nesting, each selector chosen would need to be expressly declared within the CSS for each rule set made. Nesting changes the game by allowing style rules to be "nested," where CSS selectors are structured within each other, reflecting the overall structure of the HTML written.

How to use nested CSS

Stpe 1:

Open up a file containing an HTML file and a CSS file. Here is an example HTML Page:

                
                  div class = 'container' 
                    h1 "Hello world"
                    p "Here is a paragraph"

Step 2:

Open your CSS style sheet and begin to add the class selector for .container:

                
                  .container {
                    (this is where your Nesting will go)
                  }

Step 3:

Now instead of adding rules just for the current class, define a "child" class selector within the .container class that edits h1:

                
                  .container {
                    h1{
                      (h1 is nested within the parent: .container)
                    }
                  }

Step 4:

Add the rules you want to give to h1. Note: These style rules will only be applied to the h1 within the .container class selector:

                
                  .container {
                    h1{
                      color: red;
                    }
                  }

Step 5:

Next, add a rule set for the p (paragraph) within .container:

                
                  .container {
                    h1{
                      color: red 
                    }

                    p {
                      color: blue;
                    }
                  }

Step 6:

Now you have completed the process of nesting your CSS rulesets! Go out and try it for yourself in your next CSS based work!
